String

Important Methods

charAt(int index): char
- Returns the char value at the specified index
compareTo(String anotherString): int
- Compares two strings lexicographically
- Returns 0 if equal, -ve number if less than, and +ve number if greater than
concat(String str): String
- Concatenates the specified string to the end of this string
indexOf(String str, int fromIndex): int
- Returns the index within this string of the first occurence of the specified substring, starting at the specified index
length(): int
- Returns the length of this string
replace(char oldChar, char newChar): String
- Returns a new string resulting from replacing all occurrences of oldChar in this string with newChar
split(String regex): String[]
- Splits this string around matches of the given regular expression
startsWith(String prefix): boolean
- Tests if this string starts with the specified prefix
substring(int beginIndex): String
- Returns a new string that is a substring of this string
toCharArray(): char[]
- Converts this string to a new character array
valueOf(Type<T>): static String
- Returns the string representation of the Type<T> argument
